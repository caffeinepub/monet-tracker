# Specification

## Summary
**Goal:** Rename the app to “Monet Tracker”, make the Dashboard Daily Spending chart visually distinct, and polish the UI for a more professional, cohesive finance-tracker look.

**Planned changes:**
- Update the Dashboard Weekly/Daily Spending bar chart fill color token to a different existing chart color variable to improve visual distinction while keeping theme readability.
- Replace all user-facing occurrences of the old app name with “Monet Tracker” (header, footer, Settings/About text, and HTML document title).
- Refine spacing, typography hierarchy, alignment, and subtle surface styling across the app shell and top-level screens (Dashboard, Add Expense, Expense List, Settings) without adding new features.

**User-visible outcome:** The app is branded as “Monet Tracker” throughout, the Daily Spending chart uses a distinct color, and the overall UI looks more polished and consistent in both light and dark modes.
